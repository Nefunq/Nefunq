### 🔧 What I actually build

I don't just do tutorials — every project below is cleaned, documented, and pushed with a real writeup. Click through, the code speaks for itself.

**[🌍 World Population Analysis](https://github.com/Nefunq/SCT_DS_1)**
Merged and cleaned World Bank data, then used log scaling and regional aggregation to make sense of population distribution that raw numbers hide.

**[🚢 Titanic Survival Analysis](https://github.com/Nefunq/SCT_DS_2)**
Imputed missing ages using class + gender groupings instead of dropping rows, engineered new features, and mapped out exactly who survived and why.

**[🏦 Bank Marketing Prediction](https://github.com/Nefunq/SCT_DS_3)**
Built a Scikit-learn decision tree pipeline with `ColumnTransformer` to predict term deposit subscriptions — **~89.5% accuracy**, plus real handling of class imbalance, not just a happy-path model.

**[🚗 UK Road Accident Analysis](https://github.com/Nefunq/SCT_DS_4)**
Turned raw timestamps into hourly accident patterns and built an interactive Folium hotspot map across weather, road surface, and speed limit — the kind of analysis that could actually change a policy decision.

**[💸 SplitWise Clone](https://github.com/Nefunq/splitwise-clone)**
A full PHP + MySQL expense-sharing app with transactional integrity, round-off-safe balance splitting, and settlement tracking. Not a toy — it handles money correctly.

**[🛡️ VulnScan](https://github.com/Nefunq/VulnScan)**
Consolidated Nmap scanning, Shodan enrichment, SQLi detection, and live CVE lookup into one CLI tool — replaced five manual tools and cut assessment time. Built because I got tired of switching tabs.
